# Linux Server Setup and Security

This project includes the steps to configure and secure a Linux server (Ubuntu 22.04 LTS) to host a web application.

## Steps Performed:
1. Install and configure Nginx.
2. Secure the server with UFW and SSH hardening.
3. Implement SSL with Let's Encrypt.
4. Set up automated backups.

## Technologies Used:
- Ubuntu 22.04 LTS
- Nginx
- UFW
- Let's Encrypt
- Bash scripting

## How to Use This Project:
- Clone the repository: `git clone https://github.com/username/linux-server-setup.git`
- Follow the steps in `README.md`.
